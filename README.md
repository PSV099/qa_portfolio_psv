# QA Portfolio

Welcome to my QA (Quality Assurance) Portfolio! This is a professional website showcasing my QA skills, experience, and projects.

## Structure

- **index.html** - Main portfolio page
- **css/styles.css** - Styling for the portfolio
- **js/main.js** - Interactive features
- **assets/** - Images and documents
  - profile.png - Profile picture
  - resume.pdf - Resume/CV

## Getting Started

1. Open index.html in a web browser
2. Explore the portfolio and view my projects
3. Download the resume for more details

## Features

- Responsive design
- Clean and professional layout
- Easy navigation
- Resume download option
